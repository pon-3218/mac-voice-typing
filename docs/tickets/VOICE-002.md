# VOICE-002 初回オンボーディングとログイン時起動を完成させる

## 対象アプリ

Voice Input Local

## 対象リポジトリ

`/Users/pon/dev/mac-voice-typing`

## 目的

初回起動時にFnホールド操作と必要な権限を理解でき、以後はMacへのログイン時に自動で待機状態になるようにする。

## 背景

ログイン時起動は既定で有効だが、初回は権限設定画面だけが開き、入力欄を選んでFnを押して話す操作が説明されない。

## やること

- 既存のログイン時起動の既定値と登録処理をテストで固定する
- 初回起動時だけ1画面のオンボーディングを表示する
- 基本操作とマイク・アクセシビリティ権限を説明する
- メニューバーからオンボーディングを再表示できるようにする

## やらないこと

- DMGコピー直後の強制起動
- 音声認識方式や入力処理の変更
- 複数ページのチュートリアル

## 完了条件

- 初期設定でログイン時起動が有効になる
- 未完了時だけオンボーディングが自動表示される
- 完了後は自動表示されず、メニューから再表示できる
- 権限設定とFnホールド操作が1画面で理解できる

## 検証方法

- 先にオンボーディング状態・表示契約の失敗テストを追加する
- `swift test`
- `swift build`
- `/Applications/Voice Input Local.app`を起動してオンボーディングとログイン項目を確認する
