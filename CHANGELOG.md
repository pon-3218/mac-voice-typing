# Changelog

## 0.1.7

- 専用キーを押している間に話した質問をCodexへ送る機能を追加
- 通常音声入力とCodex質問のキーを、設定画面で実際に押した左右の修飾キーへ変更可能にし、競合を防止
- Codexの処理中、回答、エラーを専用ウィンドウに表示

## 0.1.6

- 起動中の自動更新確認とバックグラウンド更新に対応
- メニューバーへ「アップデートを確認…」を追加
- GitHub Releaseで署名済みappcastを自動生成・公開

## 0.1.5

- Voice Processingの音声破損を回避し、通常のマイク入力で全文認識するよう復旧
- 最終認識結果を時間順に組み立て、結果列エラー時に一文字だけを成功扱いしないよう修正

## 0.1.4

- 音声入力中に他アプリの再生音を最大レベルで抑制
- Voice Processingのエコーキャンセルでマイクへの回り込みを低減
- 録音停止、失敗、アプリ終了時に通常再生へ確実に復帰

## 0.1.3

- 初回だけFnホールド操作と権限を案内するオンボーディングを追加
- メニューバーの「使い方…」から案内を再表示可能に変更
- ログイン時自動起動の既定動作をオンボーディング上でも明示

## 0.1.2

- Hardened Runtime向けAudio Input entitlementを追加
- メニューバー項目の可視状態を明示し、非表示状態から自動復元
- 開発署名Keychainの固定パスワードを廃止
- PR向けCI、依存関係ハッシュ固定、DMG内アプリ検証を追加
- プライバシー、セキュリティ、コントリビューション文書を追加
- SHA-256ファイルを任意の保存先で検証できる形式へ修正
- Release資産をドラフトへ揃えてから公開する不変リリース手順へ変更
